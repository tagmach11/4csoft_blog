사용자 접근성과 시각적 완성도를 높이다

LX2는 사용자 접근성과 시각적 완성도를 높이기 위해 통합검색 시스템과 홍보영상 기능을 전면 강화했습니다. 이번 업데이트는 검색 품질 향상과 콘텐츠 홍보 기능 개선을 중심으로 플랫폼 전반의 사용자 경험(UX)을 개선하는 것을 목표로 진행되었습니다.

홍보영상 기능 강화

2025년 11월 6일 ~ 7일

홍보영상 관리 및 재생 기능을 대폭 강화했습니다.

V

비메오 홍보영상 계정 공유

LX2 홍보영상을 통합 관리하기 위해 비메오(Vimeo) 전용 계정이 운영 환경에 공유되었습니다.

통합 관리 효과

  • 영상 업로드 일원화
  • 배너 연동 간소화
  • 팝업 재생 표준화
  • 각 페이지에서 동일한 계정으로 콘텐츠 관리
G

GLightbox 기반 팝업 적용

홍보영상 배너에는 GLightbox 기반의 영상 팝업 구조가 새롭게 적용되었습니다. 사용자가 배너를 클릭하면, 전체화면 팝업으로 즉시 영상 재생이 가능하도록 개선되었습니다.

✓

공통 CSS/JS 구조 추가

모든 템플릿에서 동일한 기능 사용 가능

✓

템플릿-A 확장 적용

'강의영상 한 컷' 영역에도 동일한 팝업 구조 적용

이를 통해 LX2는 홍보용 영상 콘텐츠의 노출 효과를 극대화하고, 브랜드 일관성을 유지하면서도 관리 효율성을 높였습니다.
