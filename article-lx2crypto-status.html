LX2Crypto 개발 현황 보고

2025년 · LX2

핵심 기능 개발 완료 및 검증 준비 단계 돌입

LX2는 자체 암호모듈 'LX2Crypto'의 핵심 기능 개발을 완료하고, 본격적인 검증 준비 단계에 돌입했습니다. 이번 보고에서는 SHA2·SHA3 해시함수, HMAC, CMAC 등 주요 암호 알고리즘 모듈의 구현이 완료되었으며, 현재 KISA 및 TTA 검증 절차에 대비한 내부 테스트가 진행 중입니다.

주요 구현 내용

국제표준 암호 알고리즘 구현 완료

SHA

SHA2 / SHA3 해시함수

Secure Hash Algorithm

데이터 무결성 검증 및 암호화 키 관리에 사용되는 국제표준 알고리즘을 적용했습니다. 이를 통해 LX2Crypto는 보다 강력한 암호화 기반을 확보했습니다.

주요 특징

  • 데이터 무결성 검증
  • 암호화 키 관리
  • 강력한 해시 충돌 방지
  • 국제표준 알고리즘 적용
HMAC

HMAC

Hash-based Message Authentication Code

메시지 인증 및 전송 무결성 검증 기능을 구현하여, 통신 구간에서 발생할 수 있는 데이터 변조를 방지합니다.

메시지 인증

송신자 확인

무결성 검증

변조 방지

통신 보안

안전한 전송

CMAC

CMAC

Cipher-based Message Authentication Code

대칭키 암호 기반의 인증 모듈로, 시스템 내부 데이터의 무결성을 보장합니다.

보안 강화 효과

대칭키 암호 기반으로 높은 보안성을 제공하며, 시스템 내부에서 처리되는 모든 데이터의 무결성을 실시간으로 검증합니다.

검증 절차 대비

LX2는 이번 모듈 구현 완료를 바탕으로, KISA(한국인터넷진흥원) 및 TTA(한국정보통신기술협회)의 암호모듈 검증 기준에 부합하도록 내부 검증을 진행하고 있습니다.

K

KISA 검증

한국인터넷진흥원 암호모듈 검증

T

TTA 검증

한국정보통신기술협회 기술 검증

검증 평가 항목

1

암호 알고리즘의 정확성

국제 표준에 부합하는 정확한 알고리즘 구현

2

성능 효율성

실제 운영 환경에서의 처리 속도와 자원 효율

3

보안 강도

암호화 수준과 공격 방어 능력 평가

LX2는 이를 통과하여 공공기관 및 기업 환경에서도 신뢰할 수 있는 보안 모듈을 제공할 계획입니다.

향후 통합 계획

LX2Crypto는 향후 LX2의 전반적인 보안 체계에 통합되어 다양한 영역에서 핵심 역할을 담당하게 될 예정입니다.

데이터 보호

학습 데이터 및 개인정보 암호화

사용자 인증

안전한 로그인 및 인증 시스템

세션 관리

세션 보안 및 무결성 검증

신뢰할 수 있는 보안 모듈로 한 걸음 더

공공기관과 기업이 신뢰할 수 있는 보안 환경을 제공합니다.

LX2 플랫폼의 보안 수준을 한 단계 더 높이겠습니다.
